# CSScript
A .NET Framework plugin for Take Command Console, 
to run simple C# scripts.

NOTE: You will need a copy of the TC-DotNetPluginHost64.dll file in order to compile the plugin.

On my system, 
TC-DotNetPluginHost64.dll is located in the C:\Program Files\JPSoft\TCMD36 folder.

This plugin was built using

Microsoft Visual Studio Community 2022 (64-bit) - Current
Version 17.14.36
on Windows 10 [Version 10.0.19045.7548]
19045        Microsoft Windows 10 Pro              64-bit          10.0.19045

22H2 

Review template.csx for examples of C# code that can be run with CSScript.
